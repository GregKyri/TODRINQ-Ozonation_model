# TODRINQ Ozonation model
This repository contains the code scripts of the TODRINQ model for the ozonation process (CT exposure, bromate formation, coliforms removal)

**Note:**  
Code and models are the property of TU Delft (Technical University of Technology). Do not distribute or use for commercial purposes.
The case study dataset is large and cannot be uploaded here. This could be sent upon request only.

---
## Predictions
To make predictions use the predictions.py schript.

## Retrain models
To retrain the models use the training.py schript.

## Predictive Models 
Final models for different outputs (CT, bromate, coliform removal) are saved in the respected folders.
